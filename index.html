ATI 2015 - Demo consumir API Rest desde HTML + Javascript + jQuery + AngularJS

Demos simples de como consumir una API Rest con HTML, JS, jQuery y AngularJS.

Todas las demos consumen la API "Ultimate Weather Forecasts": https://www.mashape.com/george-vustrey/ultimate-weather-forecasts

  • Simple HTML que consume el servicio y muestra el JSON recibido en un textarea (Javascript plano)

  • Consume el servicio y renderiza el objeto recibido en una plantilla simple HTML usando jQuery

  • Consume el servicio y renderiza el objeto recibido en una plantilla HTML usando AngularJS
